Five In a Row

The Five in a Row game, also referred to as Caro, is a Vietnamese logic game akin to Tic Tac Toe, with more intricate rules - requiring 5 consecutive moves to win and played on a 20x20 board.

In this project, I developed a full-stack web application that enables users to sign up and play against other players or an AI adversary. The application is built with a three-layer architecture comprising of React.js client, Spring Boot server, and MongoDB database.

Animated Photo Book

This Java-based application interprets text commands to create basic images using the Java Graphics Library and present them as a dynamic photo album. The project allowed me to practice using Object-Oriented Design principles and common design patterns to create a Model-View-Controller application with a Java Swing GUI.

Mayhem Heroes Program

Mayhem is an autonomous fuzz testing solution developed by ForAllSecure. Mayhem Heroes is a program that encourages college students to integrate the Mayhem solution into popular open-source projects on GitHub.

I intergrated Mayhem into the GitHub Action workflows of 15 repositories, written mainly in C and Rust.
